# User Stories

Create one folder per user story.

Example:

```text
manifest/user-stories/US-123/
  story.txt
  metadata-hints.txt
  package.xml
  metadata-analysis.txt
```

Rules:

- `story.txt` is mandatory.
- `metadata-hints.txt` is optional and accepts one metadata reference per line in the format `MetadataType:MemberName`.
- `package.xml` and `metadata-analysis.txt` are generated by `scripts/generate-package.sh`.
